
Click the Potato
================
Simple click competition built with Node.js + Express + SQLite.

How to run locally:
1. Install Node.js (v16+).
2. unzip the project.
3. in project folder, run:
   npm install
   npm start
4. Open http://localhost:3000

Files:
- server.js        (Express server + SQLite)
- package.json
- clicks.db        (created automatically when app runs)
- public/
    - index.html
    - style.css
    - script.js
    - potato.svg
    - squish.wav

Deploy:
Upload the project to GitHub and connect the repo to Render as a Web Service.
Start command: npm start
